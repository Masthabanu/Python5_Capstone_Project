# 📊 Marketing Campaign Analysis

## 📌 Overview

This project analyzes marketing campaign data to evaluate performance using Python.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📈 Key Analysis

* ROI analysis by company and location
* Conversion rate trends over time
* Engagement score distribution
* Channel performance comparison

## 📊 Visualizations

* Bar charts
* Line charts
* Scatter plots
* Heatmaps
* Pie charts

## 🎯 Insights

* Identified top-performing channels and companies
* Observed trends in conversion rates over time
* Compared campaign effectiveness across segments

## 🚀 Conclusion

This project demonstrates data analysis, visualization, and business insights generation.
